## üöÄ INSTALLATION

**Requirements:** `moon` (MoonBit CLI), `cargo` (Rust), `python 3.10+`.

1.  **Clone the Repo:**
    ```bash
    git clone https://github.com/merchantmoh-debug/moonlight-kernel.git
    cd moonlight-kernel
    ```

2.  **Awaken the Beast (Build):**
    ```bash
    # 1. Compile the MoonBit Core to Wasm
    cd core && moon build --target wasm

    # 2. Build the Rust Bridge
    cd ../bridge-rust && cargo build --release
    ```

3.  **Run the Protocol:**
    ```bash
    cargo run --release
    ```
